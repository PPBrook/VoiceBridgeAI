获取腾讯云密钥

  1. 登录 访问管理 → API 密钥，创建 SecretId / SecretKey
  2. 在 语音识别控制台 查看应用 AppId
  3. Secret 同时用于 ASR 识别与 TMT 翻译草稿，粘贴后保存即可

获取七牛 Key

  1. 登录 七牛 AI 控制台，创建 API Key
  2. 用于双引擎句末润色；重启后仍生效请写入 .env
