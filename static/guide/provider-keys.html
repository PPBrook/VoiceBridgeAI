各接口官网与密钥说明

在控制台 接口配置（/config）填写云端字段 → 保存 → 测试。 环境变量名与表单字段对应，详见 .env.example。

默认可选（无需配置）： 本地 Whisper（识别）、Argos 离线（句中 / 句末均可）— 直接在 引擎设置 下拉框选用即可。 句末也可选 不翻译（沿用句中），省一次翻译调用。

腾讯云

识别 句中 TMT 句末 TMT
官网
腾讯云
控制台

文档
实时语音识别 · 文本翻译
必填密钥
TENCENT_ASR_APP_ID — ASR 应用 AppId
TENCENT_ASR_SECRET_ID — API 密钥 SecretId

TMT_PROJECT_ID（默认 0）
备注
ASR 与 TMT 共用同一对 SecretId/SecretKey；识别与 TMT 需在控制台分别开通。
