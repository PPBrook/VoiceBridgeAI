## 功能速查

| 功能 | macOS | Windows |
|------|-------|---------|
| 观看场景 / 引擎链路 | 主窗口或 设置 → 引擎 | 同左 |
| 本地模型 | 设置 → 本地模型（cloud 无 Tab） | 同左 |
| 接口密钥 | 设置 → 接口密钥 | 同左 |
| 字幕记录 | 设置 → 字幕记录；悬浮「记」 | 同左 |
| 打开记录目录 | 菜单栏 | 托盘 |
| 透明度 / EN | 悬浮顶栏 | 同左 |
